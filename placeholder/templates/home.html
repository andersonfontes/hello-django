Images Placeholder Django

Este servidor pode ser usado para servir imagens de Placeholder (guardador de lugar) nos tamanhos que voce precisar para qualquer web page

Para requisitar uma imagem placholder, de um determindado altura e largura, simplesmente inclua uma imagem com o caminho apontando para /image/<width>/ neste servidor como: 

            <img src="{{ example }}" >
